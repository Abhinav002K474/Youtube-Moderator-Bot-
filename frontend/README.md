# 📺 YouTube Moderator Bot - Frontend

This is the frontend dashboard for the YouTube Moderator Bot. It is built with **React 19** and **Vite**.

## 🚀 Features
- Real-time chat feed with moderation flags.
- Live analytics cards (viewers, likes, subscribers).
- Interactive moderation actions (Delete, Timeout).
- Settings dashboard for bot configuration.

## 🛠️ Setup
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`

For full project details and backend setup, please refer to the [Root README](../README.md).
